# Retail Sales Analytics

Análisis exploratorio y modelo predictivo de ventas retail con Python.

## Stack
pandas, numpy, matplotlib, seaborn, scipy, scikit-learn

## Contenido
- Limpieza y transformación de datos
- Feature engineering desde fechas
- Análisis de ventas y profit por categoría
- Impacto del descuento en el margen
- Estacionalidad mensual
- Test estadístico ANOVA entre regiones
- Modelo predictivo de ventas con regresión lineal

## Hallazgos principales
- Furniture lidera en ventas pero genera pérdidas (-$100K)
- Descuentos sobre 10% destruyen el profit
- No hay diferencia estadísticamente significativa entre regiones (p=0.81)
- Las ventas muestran estacionalidad anual con pico en Q4
